# Installation

## Complete library

<br>

<code-group>
  <code-block title="Vue">
  ```bash
  yarn add @naked-ui/vue
  ```
  </code-block>

  <code-block title="HTML">
  ```html
  <script src="https://cdn.jsdelivr.net/npm/naked-ui"></script>
  ```
  </code-block>
</code-group>

<br>

<br>

<code-group>
